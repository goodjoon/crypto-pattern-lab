# Project Charter — Crypto Pattern Lab

## 1. 왜 이 프로젝트를 하는가

Crypto Pattern Lab은 코인 시장의 거래 패턴과 온체인 패턴을 연구하고, 실제 수익 가능성이 있는 신호를 검증한 뒤, 이후 자동 실행 시스템으로 연결하기 위한 기반 프로젝트다.

핵심 목적은 단순한 자동매매 봇 제작이 아니다. 먼저 다음을 만족하는 연구 시스템을 만드는 것이다.

- 어떤 패턴이 반복되는가
- 어떤 패턴이 실제 수익 가능성이 있는가
- 비용과 리스크를 반영해도 유효한가
- 실시간 운영 시스템으로 확장 가능한가

## 2. 프로젝트 원칙

- 전략 없는 실행기는 만들지 않는다.
- 분석 → 가설 → 검증 → 자동화 순서를 지킨다.
- 수수료, 슬리피지, 체결 제약, API 제한을 반드시 반영한다.
- 백테스트 결과만으로 신뢰하지 않고 워크포워드/페이퍼 테스트를 거친다.
- 재현 가능한 연구 환경과 버전 관리된 문서를 유지한다.

## 3. 1차 범위

초기에는 범위를 너무 넓히지 않는다.

- 중앙화 거래소 1~2개
- 이더리움 계열 온체인 데이터 일부
- 대상 마켓/토큰 소수
- 패턴 탐지 및 신호 검증 파이프라인 우선

## 4. 기대 산출물

- 데이터 수집 파이프라인
- 공통 이벤트 스키마
- 패턴 후보 라이브러리
- 백테스트/리포트 파이프라인
- 대시보드 초안
- 실행 시스템으로 확장 가능한 구조
