# Crypto Pattern Lab — MVP Scope v0.1

## 1. 범위

### 거래소
- Upbit
- Coinone
- Binance
- Bybit

### 온체인
- Ethereum mainnet
- ERC-20 토큰
- 주요 Ethereum L2 체인(후보: Arbitrum, Optimism, Base)

## 2. MVP에서 먼저 풀 문제

1. 국내/해외 거래소 간 가격/체결 패턴 차이
2. 동일 자산의 거래소별 반응 속도 차이
3. 온체인 대규모 이동과 거래소 가격 반응 관계
4. 거래소 입출금/온체인 이벤트가 단기 가격 변화에 미치는 영향

## 3. 1차 우선순위

### 우선순위 A — 시장 데이터 기반
- 거래소 티커/체결/기본 호가 수집
- 공통 심볼 매핑
- KRW 마켓 vs USDT 마켓 비교
- 괴리 추적 및 리포트

### 우선순위 B — 온체인 이벤트 기반
- ERC-20 transfer 수집
- 거래소 핫월렛/주요 주소 후보 태깅
- 대규모 이동 탐지
- 이벤트 이후 가격 반응 측정

### 우선순위 C — 하이브리드 분석
- 온체인 대규모 입금 후 거래소 가격 반응
- 특정 거래소 체결 집중도와 온체인 흐름의 결합 분석

## 4. 제외 범위

MVP에서는 아래를 바로 하지 않는다.

- 실거래 주문 실행
- 고빈도/초저지연 전략
- 다수 체인 동시 확장
- 복잡한 모델링 기반 예측
- 대규모 대시보드 제품화

## 5. 선행 리서치 필요 항목

사용자는 2018년 PoW 시절까지의 Ethereum 경험은 강하지만, 현재 Ethereum/L2/최신 토큰 생태계는 재정리가 필요하다. 따라서 MVP 초기에는 아래 문서를 병행한다.

- 최신 Ethereum 생태계 요약
- 주요 L2 비교
- 거래소/온체인 데이터 소스 비교
- 연구 대상 토큰/마켓 후보 리스트

## 6. 바로 다음 단계

1. 거래소 API 제약 확인
2. 공통 심볼/마켓 매핑 문서 작성
3. Ethereum/L2 최신 생태계 브리프 작성
4. 수집 대상 이벤트 스키마 초안 작성
5. 초기 프로젝트 코드 골격 생성
