# Crypto Pattern Lab

코인 거래/트랜잭션 패턴을 수집, 분석, 검증하고 이후 자동매매 시스템으로 확장하기 위한 연구 프로젝트.

## 목표

1. 거래소/온체인 데이터를 수집한다.
2. 반복되는 거래 패턴과 알파 신호 후보를 탐지한다.
3. 수수료/슬리피지/체결 현실을 반영해 전략을 검증한다.
4. 검증된 전략만 실행 시스템으로 연결한다.
